# Product Overview

## Jobs & Employment Platform

A modern job search and employment platform built with Next.js that connects job seekers with companies. The platform provides a comprehensive job search experience with company profiles, job listings, and user profiles.

## Core Features

- **Job Search & Discovery**: Browse and search job listings with filtering capabilities
- **Company Profiles**: View detailed company information, ratings, reviews, and open positions
- **User Authentication**: Secure login/register system with NextAuth integration
- **User Profiles**: Job seeker profiles with experience and education sections
- **Dashboard**: User dashboard for managing activities, bookmarks, and settings
- **Internationalization**: Multi-language support (English/Arabic) with RTL support

## Target Users

- **Job Seekers**: Browse jobs, view companies, manage applications
- **Companies**: Post jobs, manage company profiles
- **Admins**: Platform administration and user management

## Key User Flows

1. Browse jobs and companies as guest
2. Register/login to apply for jobs
3. Build and manage user profile
4. Bookmark and track job applications
5. View company details and reviews
